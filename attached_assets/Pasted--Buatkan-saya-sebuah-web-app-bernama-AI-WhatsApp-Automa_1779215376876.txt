
Buatkan saya sebuah web app bernama “AI WhatsApp Automation Assistant” yang dapat membalas semua chat WhatsApp secara otomatis selama 24 jam menggunakan AI, dengan kemampuan memahami produk knowledge dan menjawab seperti customer service profesional yang ramah, cepat, dan membantu closing.

Tujuan utama:
Menggantikan admin WhatsApp manual menjadi AI yang:

Bisa menjawab pertanyaan customer
Bisa menjelaskan produk
Bisa mengarahkan ke pembelian
Bisa tetap sopan dan engaging

TECH STACK:

Frontend: React + Next.js + Tailwind
Backend: Node.js (Express)
WhatsApp Integration: WhatsApp Web API (gunakan library seperti Baileys atau venom-bot)
AI:

OpenAI GPT (untuk balasan)
Embedding / vector DB (untuk knowledge base, gunakan Pinecone / Supabase vector / local vector store)

Database:

PostgreSQL / Firebase

FITUR UTAMA:

KONEKSI WHATSAPP (WAJIB ADA)
Login menggunakan QR Code (scan seperti WhatsApp Web)
Status:
Connected
Disconnected
Auto reconnect jika terputus
AUTO REPLY 24 JAM
Semua chat masuk akan dibalas otomatis oleh AI
Delay natural (1–3 detik) agar terasa manusiawi
Bisa handle:
chat baru
follow up chat lama
KNOWLEDGE BASE (FITUR PALING PENTING)

User bisa input:

Deskripsi produk
FAQ (QnA)
Link website
Script jualan
Testimoni

AI harus belajar dari data ini menggunakan sistem embedding (RAG).

AI RESPONSE ENGINE

Gunakan prompt system seperti ini:

“Kamu adalah customer service profesional yang ramah, cepat, dan membantu closing.

Gunakan gaya bahasa:

Santai tapi sopan
Gunakan emoji secukupnya 😊🔥

Gunakan knowledge berikut untuk menjawab:
[DATA KNOWLEDGE BASE]

[CHAT USER]

Tugas kamu:

Jawab pertanyaan dengan jelas
Jika memungkinkan, arahkan ke pembelian

Jangan jawab di luar knowledge
Jika tidak tahu, arahkan ke admin manusia”
FALLBACK KE HUMAN AGENT (PENTING BANGET)

AI harus bisa:

Jelaskan benefit

DASHBOARD ADMIN
Lihat semua chat
Status:
